# 计算机组成原理概论

## 计算机分类和性能指标
**计算机指令：** 它从存储器流入控制器，并变成整个计算机系统的控制信号。
**数据：** 从输入设备流入存储器，再流入执行部件如运算器，运算结果流入存储器或输出设备。
**分类方法及结果**
1. 计算机体系结构分类：（1）SISD -- 单指令单数据 （2）SIMD -- 单指令多数据 （3）MISD -- 多指令单数据 （4）MIMD -- 多指令多数据
2. 计算机用途分类：（1）通用计算机 （2）专用计算机
3. 使用类型分类：（1）桌上型计算机 （2）服务器型计算机 （3）嵌入式计算机
4. 计算机的规模分类：（1）巨型计算机 （2）大/中型计算机 （3）小型计算机 （4）微型计算机

**性能指标：**
1. **机器字长：** CPU一次能处理的数据位数
2. **存储容量：** 存储单元个数 × 存储字长
3. **运算速度（MIPS、CPI、FLOPS）**
4. **可配置的外设：**
5. **性能价格比**
6. **可靠性（MTBF）、可维修性（MTRF）和可用性（A）**

## 计算机的基本组成
**计算机硬件系统五大部件：** 运算器、存储器、控制器、输入设备、输出设备。

**系统总线：** 地址总线(AB)、数据总线(DB)、控制总线(CB)。
